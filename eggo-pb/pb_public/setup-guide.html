⚠️ OAuth Setup Required

Fix "Failed to create record" error by configuring OAuth in Admin UI

🚨 Problem: OAuth field mapping is not configured.
Solution: Takes 2 minutes in Admin Panel.

Step 1: Open Admin Panel

Open Admin Panel →
Email: admin@eggo.local
Password: admin123

Step 2: Go to OAuth Settings

Settings → Collections → users → Click OAuth2 tab

Step 3: Add/Edit OAuth Provider

📌 Note: If you see "oidc" already listed, click Edit (✏️). Otherwise click "Add Provider".

Configure these settings:

Name: oidc
Display Name: Line
Auth URL: https://access.line.me/oauth2/v2.1/authorize
Token URL: https://api.line.me/oauth2/v2.1/token
Fetch user info from: User info URL
User info URL: https://api.line.me/oauth2/v2.1/userinfo
Support PKCE: ✓ (checked)

Step 4: Field Mapping ⭐ MOST IMPORTANT

Click "Optional users create fields map" to expand

Map these fields (select from dropdown):

OAuth2 Field PocketBase Field
OAuth2 id → externalId
OAuth2 full name → name
OAuth2 avatar → avatar

✓ Select from dropdown menus - these are the exact field names in the PocketBase UI.

Step 5: Save

Scroll down and click Save button
